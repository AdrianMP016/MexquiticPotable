1 ¿Qué es el sistema?

El Sistema de Agua Potable de Mexquitic es una plataforma web que permite administrar de forma digital todo el proceso de cobro de agua: desde registrar a los usuarios y sus medidores, hasta capturar lecturas en campo, generar recibos y registrar pagos.

🖥️

Panel Admin

Para el personal de oficina. Gestiona usuarios, medidores, recibos y pagos.

📋

Verificador

Para el lector de campo. Captura las lecturas de los medidores en casa de cada usuario.

💳

Cobro en campo

Para el cobrador. Escanea el QR del recibo y registra el pago en el momento.

💡 Cada módulo tiene su propio acceso y contraseña. Un verificador no puede ver el panel de administración y viceversa.
2 Cómo ingresar al sistema

Existen tres páginas de acceso, una por cada módulo:

MóduloURL de accesoPara quién
Administración/login-admin.phpDirectivos y personal de oficina
Verificador/login-verificador.phpLectores de medidores en campo
Cobro/login-cobro.phpCobradores en campo

Pasos para ingresar

  1. Abre el navegador y escribe la dirección del sistema.
  2. Ingresa tu usuario y contraseña asignados por el administrador.
  3. Presiona Iniciar sesión.
  4. Si es la primera vez, el administrador debe crearte una cuenta desde Usuarios del sistema.
⚠️ Si olvidaste tu contraseña, contacta al administrador del sistema. No compartas tu contraseña con nadie.
3 Panel de Administración

Al ingresar como administrador verás un menú lateral izquierdo con todas las secciones del sistema.

3.1 Alta de usuario

Permite registrar a un nuevo usuario en el padrón de agua.

  1. Haz clic en Alta de usuario en el menú.
  2. Llena los datos del usuario: nombre completo, teléfono, WhatsApp.
  3. Selecciona la ruta a la que pertenece.
  4. Agrega su domicilio: calle, número, colonia.
  5. Opcionalmente puedes marcar su ubicación en el mapa.
  6. Presiona Guardar. El sistema revisará si ya existe un usuario similar para evitar duplicados.
📝 Después de dar de alta al usuario, debes asignarle un medidor desde la sección Medidores.

3.2 Consulta de usuarios

Muestra la lista completa de todos los usuarios registrados en el padrón.

  • Puedes buscar por nombre, ruta o número de medidor.
  • Desde aquí puedes editar los datos de un usuario o darlo de baja.
  • También puedes ver si tiene medidor asignado y cuál es su ruta.

3.3 Medidores

Aquí se administran los medidores de agua asignados a cada usuario.

  • Cada usuario debe tener exactamente un medidor activo para poder capturar lecturas.
  • El número de medidor sigue el formato SAPMXQ-XXXX (ej. SAPMXQ-0507).
  • Si un medidor se reemplaza, se marca como reemplazado y se crea el nuevo.
Estado del medidorSignificado
ActivoEn uso normal, acepta lecturas
InactivoSuspendido temporalmente
ReemplazadoSe cambió por uno nuevo
Sin medidorEl usuario no tiene medidor instalado

3.4 Rutas

Las rutas agrupan a los usuarios por zona geográfica para organizar el trabajo del verificador.

  • Cada ruta tiene un código (ej. Bellav-001) y un nombre descriptivo.
  • El verificador de campo busca usuarios seleccionando una ruta.
  • Desde aquí puedes crear nuevas rutas, editarlas o darlas de baja.

3.5 Periodos bimestrales

El cobro del agua se realiza cada dos meses (bimestre). Los periodos definen las fechas de cada ciclo de cobro.

PeriodoMesesEjemplo 2026

💡 El sistema detecta automáticamente qué periodo está vigente según la fecha actual. No es necesario activarlo manualmente.

3.6 Lecturas y recibos

Esta sección es el núcleo del proceso de cobro. Aquí puedes:

  • Ver todas las lecturas capturadas por el verificador para cada periodo.
  • Generar recibos individuales o por periodo completo.
  • Imprimir recibos en lote para repartirlos.
  • Enviar recibos por WhatsApp directamente al usuario.

¿Cómo generar un recibo?

  1. Busca al usuario por nombre o ruta.
  2. Selecciona el periodo del que quieres generar el recibo.
  3. El sistema calcula automáticamente el monto según el consumo.
  4. Haz clic en Generar recibo. Se crea con un código QR único.
  5. Puedes enviarlo por WhatsApp o imprimirlo.
📝 Solo se puede generar recibo si el verificador ya capturó la lectura del periodo correspondiente.

3.7 Pagos

Permite registrar y consultar los pagos realizados por los usuarios.

  • Busca al usuario y verás sus recibos pendientes de pago.
  • Selecciona el recibo y registra el pago manualmente.
  • También puedes eliminar un pago registrado por error.
  • Se lleva registro de quién registró cada pago y a qué hora.
💡 El cobrador en campo puede registrar pagos directamente escaneando el QR del recibo desde el módulo Cobro, sin necesidad de entrar al panel admin.

3.8 WhatsApp

El sistema puede enviar notificaciones y recibos automáticamente por WhatsApp.

  • Desde aquí se configura la sesión de WhatsApp del sistema.
  • Se pueden enviar notificaciones masivas a todos los usuarios de un periodo.
  • Se puede ver el historial de mensajes enviados.
⚠️ Si el WhatsApp se desconecta, el administrador debe reconectarlo desde esta pantalla.

3.9 Usuarios del sistema

Solo visible para el administrador. Aquí se crean las cuentas de acceso para el personal.

RolLo que puede hacer
AdminTodo: crear usuarios del sistema, ver bitácora, configurar todo
SupervisorConsultar, generar recibos, registrar pagos, pero no puede eliminar datos ni gestionar cuentas

📝 La bitácora registra automáticamente todas las acciones del sistema: quién hizo qué y cuándo.
4 Módulo Verificador — Lectura en campo

Esta es la herramienta que usa el lector de medidores cuando va de casa en casa. Está diseñada para funcionar en un teléfono celular.

Flujo de trabajo del verificador

1. Iniciar sesión
→
2. Seleccionar ruta
→
3. Buscar usuario
→
4. Ver detalle
→
5. Capturar lectura
→
6. Guardar y continuar

Paso a paso

  1. Ingresar: Abre el navegador en el teléfono y entra con tu usuario y contraseña de verificador.
  2. Seleccionar ruta: En la pantalla principal elige la ruta que te fue asignada (ej. Bellav-001, Cent1-095).
  3. Ver lista de usuarios: Aparece la lista de todos los usuarios de esa ruta.
  4. Seleccionar usuario: Toca el nombre del usuario cuyo medidor vas a leer.
  5. Ver detalle: Verás el nombre, dirección, número de medidor y la última lectura registrada (referencia para calcular el consumo).
  6. Registrar lectura: Presiona "Registrar lectura" e ingresa:
    • El número que muestra el medidor físico actualmente.
    • Opcionalmente: foto del medidor, observaciones, ubicación GPS.
  7. Guardar: El sistema calcula automáticamente el consumo. Presiona "Guardar lectura".

Información que muestra el detalle del usuario

CampoDescripción
MedidorNúmero de medidor (ej. SAPMXQ-0510)
RutaCódigo de ruta del usuario
DirecciónDomicilio del usuario
Ultima lecturaLo que marcaba el medidor al inicio de este periodo
Periodo de cobroBimestre activo (ej. Mayo-Junio 2026)
Rango del periodoFechas de inicio y fin del bimestre
💡 Si ya capturaste la lectura de un usuario, al volver a entrar verás el valor que guardaste como "Lectura actual guardada".
📝 Si hay dos usuarios con el mismo nombre, identifícalos por el número de medidor o la dirección.
5 Módulo Cobro en campo

Herramienta para el cobrador que visita a los usuarios para recibir el pago de sus recibos.

Paso a paso

  1. Ingresa con tu usuario y contraseña de cobrador.
  2. Para cobrar un recibo tienes dos opciones:
    • Escanear QR: El usuario presenta su recibo (impreso o en pantalla) y el cobrador escanea el código QR con la cámara del teléfono.
    • Buscar manualmente: Escribe el nombre del usuario y selecciona el recibo pendiente.
  3. El sistema muestra el recibo con el monto a pagar.
  4. Confirma el pago. Queda registrado de inmediato con hora y nombre del cobrador.
⚠️ No registres un pago que no hayas recibido físicamente. Los pagos quedan registrados con tu nombre de usuario.
6 Fórmula de cobro de agua

El costo del agua se calcula de la siguiente manera:

ConsumoPrecio por m³
Primeros 15 m³$10.00 por m³
A partir del m³ 16 en adelante$15.00 por m³
Si consumo ≤ 15 m³: Total agua = consumo × $10 Si consumo > 15 m³: Total agua = ($10 × 15) + ($15 × metros excedentes)

Ejemplo práctico

Un usuario que consumió 20 m³ en el bimestre:

  • Primeros 15 m³ → 15 × $10 = $150
  • 5 m³ excedentes → 5 × $15 = $75
  • Total de agua: $225

Cargos fijos por bimestre

ConceptoMonto
Cooperación bimestral$10.00 (fijo, todos los usuarios)
Multa por incumplimientoSegún aplique
Recargo por pago tardíoSegún aplique
📝 La cooperación de $10 se cobra en todos los bimestres independientemente del consumo. Los demás cargos los asigna el administrador según cada caso.
7 Trabajo sin conexión (modo offline)

El verificador puede trabajar sin internet en zonas sin señal. Para esto debes preparar el dispositivo antes de salir a campo.

Cómo prepararse para trabajar sin conexión

  1. Asegúrate de tener conexión a internet antes de salir.
  2. Entra al verificador en el teléfono.
  3. Presiona el botón "Preparar para campo" (ícono de descarga en la barra superior).
  4. Espera a que descargue todos los usuarios y sus datos. Aparecerá el mensaje: "Listo. XXX usuarios disponibles sin conexión".
  5. Ya puedes salir a campo. El teléfono funcionará sin internet.

¿Qué pasa con las lecturas capturadas sin internet?

  • Las lecturas se guardan temporalmente en el teléfono.
  • En la barra superior aparece el número de lecturas pendientes de sincronizar.
  • Cuando recuperes conexión, presiona "Sincronizar" y las lecturas se enviarán al servidor.
💡 Siempre sincroniza antes de terminar la jornada y antes de que otro verificador use el mismo teléfono.
⚠️ Si borras el historial del navegador o cierras la app completamente sin sincronizar, podrías perder las lecturas guardadas localmente.
8 Preguntas frecuentes

¿Por qué no aparece un usuario en el verificador?

Posibles causas: el usuario está marcado como inactivo, no tiene medidor asignado, o no está en la ruta que seleccionaste.

¿Qué significa "Ultima lectura: 0"?

El medidor no tiene lecturas anteriores registradas. El verificador debe capturar el número actual del medidor físico normalmente.

¿Puedo corregir una lectura ya guardada?

Desde el verificador puedes volver a entrar al mismo usuario en el mismo periodo y el sistema actualizará la lectura. Si el recibo ya fue generado, el administrador debe eliminarlo y regenerarlo.

¿Por qué sale el mensaje "Sin conexión" al entrar al detalle de un usuario?

No presionaste "Preparar para campo" antes de salir sin internet. Regresa a una zona con señal y descarga los datos primero.

¿Qué hago si el medidor está dañado o no se puede leer?

Usa el campo Observaciones para describir la situación (ej. "Medidor dañado", "Casa cerrada", "No se puede leer"). Puedes guardarlo sin capturar la lectura numérica.

¿Cómo sé que la sincronización funcionó?

El contador de "pendiente(s)" en la barra superior debe quedar en 0 después de sincronizar.

¿Un usuario puede tener dos cuentas?

Sí. Si una persona tiene dos propiedades con medidor, aparece dos veces con distinto número de padrón y diferente ruta/medidor.

¿Cómo agrego un medidor nuevo a un usuario?

En el panel de administración, entra a Medidores, busca el usuario y usa el botón "Agregar medidor". El número debe seguir el formato SAPMXQ-0000.
