1 ¿Qué es el sistema?

El Sistema de Agua Potable de Mexquitic es una plataforma web que permite administrar de forma digital todo el proceso de cobro de agua: desde registrar usuarios y medidores, hasta capturar lecturas en campo, generar recibos y registrar pagos.

🖥️

Panel Admin

Para el personal de oficina. Gestiona usuarios, medidores, recibos y pagos.

📋

Verificador

Para el lector de campo. Captura las lecturas de los medidores en casa de cada usuario.

💳

Cobro en campo

Para el cobrador. Escanea el QR del recibo y registra el pago en el momento.

💡 Cada módulo tiene su propio acceso. Un verificador no puede ver el panel de administración y viceversa.
2 Cómo ingresar al sistema

Hay tres páginas de acceso, una por cada módulo. Todas están en la misma pantalla, seleccionando la pestaña correspondiente:

🖼️Login Administrador
🔵 Login — Administrador
🖼️Login Cobro
💚 Login — Cobro
🖼️Login Verificador
🟠 Login — Verificador

Pasos para ingresar

  1. Abre el navegador y escribe la dirección del sistema.
  2. Selecciona tu módulo (Administrador / Cobro / Verificador) en las pestañas.
  3. Ingresa tu usuario y contraseña.
  4. Presiona Entrar.
⚠️ Si olvidaste tu contraseña usa la pestaña "Recuperar contraseña". No compartas tu contraseña con nadie.
3 Panel de Administración
🖼️Pantalla principal del panel de administración
📊 Panel principal — Consulta de usuarios del padrón

Al ingresar como administrador verás un menú lateral con todas las secciones del sistema.

3.1 Alta de usuario

🖼️Pantalla: Alta de usuario
➕ Alta de usuario — formulario de registro
  1. Llena nombre completo, teléfono y WhatsApp del usuario.
  2. Selecciona la ruta a la que pertenece.
  3. Agrega su domicilio: calle, número, colonia.
  4. Opcionalmente marca su ubicación en el mapa.
  5. Presiona Guardar. El sistema revisará duplicados.
📝 Después de dar de alta al usuario, asígnale un medidor desde la sección Medidores.

3.2 Consulta de usuarios

Muestra la lista completa del padrón. Puedes buscar por nombre, ruta o medidor, y editar o dar de baja usuarios desde aquí.

3.3 Medidores

🖼️Pantalla: Medidores
⚙️ Gestión de medidores
  • Cada usuario debe tener un medidor activo para capturar lecturas.
  • El número de medidor sigue el formato SAPMXQ-0000.
  • Si un medidor se reemplaza, se marca como reemplazado y se crea el nuevo.

3.4 Rutas

🖼️Pantalla: Rutas
🗺️ Gestión de rutas de cobranza

Las rutas agrupan usuarios por zona geográfica. El verificador busca usuarios seleccionando una ruta (ej. Bellav-001).

3.5 Periodos bimestrales

🖼️Pantalla: Periodos bimestrales
📅 Periodos de cobro bimestral 2026

El cobro del agua se realiza cada dos meses. Hay 6 periodos por año. El sistema detecta automáticamente el periodo activo según la fecha.

BimestreMeses
1Enero – Febrero
2Marzo – Abril
3Mayo – Junio
4Julio – Agosto
5Septiembre – Octubre
6Noviembre – Diciembre

3.6 Lecturas y recibos

🖼️Pantalla: Lecturas y Recibos
📄 Lecturas capturadas y generación de recibos
  1. Busca al usuario por nombre o ruta.
  2. Selecciona el periodo.
  3. El sistema calcula el monto automáticamente según el consumo.
  4. Haz clic en Generar recibo. Se crea con código QR único.
  5. Envíalo por WhatsApp o imprímelo.
📝 Solo se puede generar recibo si el verificador ya capturó la lectura del periodo.

3.7 Pagos

🖼️Pantalla: Pagos
💰 Registro y consulta de pagos
  • Busca al usuario y verás sus recibos pendientes.
  • Selecciona el recibo y registra el pago manualmente.
  • Se guarda quién registró cada pago y a qué hora.

3.8 WhatsApp

🖼️Pantalla: WhatsApp
💬 Configuración e integración con WhatsApp
  • Envío de recibos y notificaciones masivas por WhatsApp.
  • Se puede ver el historial de mensajes enviados.
⚠️ Si el WhatsApp se desconecta, el administrador debe reconectarlo desde esta pantalla.
4 Módulo Verificador — Lectura en campo
🖼️Verificador en teléfono
📱 Módulo verificador — vista en smartphone

Herramienta para el lector de medidores en campo. Diseñada para teléfono celular.

Flujo de trabajo

1. Iniciar sesión
→
2. Seleccionar ruta
→
3. Elegir usuario
→
4. Ver detalle
→
5. Capturar lectura
→
6. Guardar

Paso a paso

  1. Ingresa con tu usuario y contraseña de Verificador.
  2. Selecciona la ruta asignada (ej. Bellav-001).
  3. Aparece la lista de usuarios de esa ruta. Toca el nombre del usuario cuyo medidor vas a leer.
  4. Verás el número de medidor, dirección y la última lectura registrada.
  5. Presiona "Registrar lectura" e ingresa el número actual del medidor. Opcionalmente: foto del medidor, observaciones y GPS.
  6. Presiona "Guardar lectura". El consumo se calcula automáticamente.

MedidorNúmero de medidor (ej. SAPMXQ-0510)
Ultima lecturaLo que marcaba el medidor al inicio de este bimestre
Periodo de cobroBimestre activo (ej. Mayo-Junio 2026)
Rango del periodoFechas de inicio y fin del bimestre
💡 Si ya capturaste la lectura de un usuario, al volver a entrar verás el valor guardado como "Lectura actual guardada".
5 Módulo Cobro en campo
🖼️Cobro en campo en teléfono
📱 Módulo cobro — vista en smartphone
  1. Ingresa con tu usuario y contraseña de Cobrador.
  2. Para cobrar un recibo:
    • Escanear QR: El usuario presenta su recibo y el cobrador escanea el código QR.
    • Buscar manualmente: Escribe el nombre del usuario y selecciona el recibo.
  3. El sistema muestra el monto a pagar. Confirma el pago.
⚠️ Los pagos quedan registrados con tu nombre de usuario. No registres un pago que no hayas recibido.
6 Fórmula de cobro de agua
ConsumoPrecio por m³
Primeros 15 m³$10.00 / m³
A partir del m³ 16$15.00 / m³
Si consumo ≤ 15 m³: Total = consumo × $10 Si consumo > 15 m³: Total = (15 × $10) + (excedente × $15)

Ejemplo: 20 m³ consumidos

  • 15 m³ × $10 = $150
  • 5 m³ × $15 = $75
  • Total: $225 + $10 cooperación = $235
📝 La cooperación bimestral de $10 se cobra a todos los usuarios independientemente del consumo.
7 Trabajo sin conexión (modo offline)
  1. Con internet disponible, entra al verificador en el teléfono.
  2. Presiona "Preparar para campo" (botón de descarga en la barra superior).
  3. Espera el mensaje: "Listo. XXX usuarios disponibles sin conexión".
  4. Ya puedes salir a campo sin internet.
  • Las lecturas capturadas sin internet se guardan en el teléfono.
  • Al recuperar señal, presiona "Sincronizar" para enviarlas al servidor.
⚠️ Siempre sincroniza antes de terminar la jornada. No borres el historial del navegador con lecturas pendientes.
8 Preguntas frecuentes

¿Por qué no aparece un usuario en el verificador?

Posibles causas: el usuario está inactivo, no tiene medidor asignado, o no está en la ruta seleccionada.

¿Qué significa "Ultima lectura: 0"?

El medidor no tiene lecturas anteriores. El verificador debe capturar el valor actual del medidor físico normalmente.

¿Puedo corregir una lectura ya guardada?

Sí. Entra al mismo usuario en el mismo periodo y el sistema actualizará la lectura. Si el recibo ya fue generado, el administrador debe eliminarlo y regenerarlo.

¿Qué hago si el medidor está dañado o no se puede leer?

Usa el campo Observaciones para describirlo (ej. "Medidor dañado", "Casa cerrada"). Puedes guardar sin capturar lectura.

¿Cómo sé que la sincronización funcionó?

El contador de "pendiente(s)" en la barra superior queda en 0 después de sincronizar.

¿Un usuario puede aparecer dos veces?

Sí, si tiene dos propiedades con medidores distintos. El sistema las trata como cuentas separadas, cada una con su propio padrón.
